FSImportTool

  This Zope Product is part of the Rhaptos system
  (http://software.cnx.rice.edu)

  FSImportTool provides a CMF tool for recursively importing and
  exporting content between the filesystem and the ZODB.  It uses the
  content_type_registry to determine the correct portal_type for
  content.

Future plans

  - This tool will be replaced by FSDumpTool:
    http://software.cnx.rice.edu/downloads/zope/FSDumpTool
